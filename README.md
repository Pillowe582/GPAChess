# 自走棋：超破防期末周！
## 游戏介绍
- 你说的对，但是《超破防期末周！》是由Pillowe自主研发的一款全新自走棋冒险游戏。游戏发生在一个被称作「大学」的真实世界，在这里，被招生组选中的学生将被授予「课程表」，挑战知识之力。你将扮演一位名为「考生」的神秘角色，在接踵而至的挑战中邂逅性格各异、能力独特的同伴们，和他们一起通过考试，找回失散的GPA——同时，逐步发掘「期末周」的真相。
## 游戏机制
### 1. 选课
- 游戏开始前，玩家需要填写自己本学期计划修读的课程数目，之后由系统随机生成各科目学分数
- 提交选课后（可能会掉课），可以选择保留结果开始游戏，或是再次刷新课程。
### 2. 玩家属性
#### 金币
- 获取方式
    - 每局结束后获得固定数量
    - 击败敌方单位有概率额外获得
    - 出售角色100%返还
- 用途
    - 购买角色
    - 刷新商店
#### 经验
- 获取方式
    - 击败敌方单位必定获得
- 用途
  - 经验用于玩家等级升级
#### 玩家等级
- 用途
  - 随等级升高出战角色上限提高
### 3. 流程
- 回合总数等于选课数目，每回合分为准备阶段和战斗阶段
- 达到回合总数时，游戏结束，结算
#### 准备阶段
- 玩家可以自由在商店购买角色、出售角色、在部署区放置角色
- 点击“开始回合”按钮进入战斗阶段
#### 战斗阶段
- 商店、部署区皆锁定，场上随机刷新敌方角色，自动开始对战
- 达成战斗结束条件时，结算奖励并结束回合，并进入下一回合的准备阶段
### 4. 敌方单位
- 必修科目：
  - 战斗开始时一次性生成一轮
  - 优先攻击玩家角色，玩家角色被全部消灭后，攻击成绩单
- 选修科目：
  - 上一轮敌方单位全灭且玩家角色未全灭时，生成一轮选修单位
  - 最多生成n轮，n与当前回合编号、当前回合学分数有关
  - 不攻击成绩单
### 5. 我方单位
- 成绩单
  - 显示本回合学分绩
  - 每回合开始时刷新到场地最左侧，学分绩重置到4.0
  - 按照自身血量百分比发射激光攻击必修科目
  - 免疫选修科目伤害
- 玩家角色
### 6. 回合结束条件
- 胜利条件
  - 击败所有必修科目与选修科目
  - 在所有必修科目已被击败的前提下，玩家角色被选修科目全部击败
- 失败条件
  - 成绩单学分绩降至0
